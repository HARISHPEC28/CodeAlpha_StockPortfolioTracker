# Stock Portfolio Tracker (Python)

A simple Python program that allows users to view available stocks, purchase multiple stocks, calculate totals, and save the transaction into a CSV file.

## Features
- Displays all available stocks with their prices
- Allows purchasing multiple stocks in one session
- Validates stock name and quantity inputs
- Calculates total purchase cost
- Saves all transactions into a CSV file (purchace.csv)

## How to Run
1. Install Python 3 on your system.
2. Save the program code as `portfolio_tracker.py`.
3. Run the script using:
   python portfolio_tracker.py
4. After finishing, the CSV file will be created or updated automatically.
